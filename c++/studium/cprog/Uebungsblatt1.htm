Aufgabe 01

Lösung per e-mail an hiwi509@btcipmatx1.cip.uni-bayreuth.de (Bernd Seifert)

Geben Sie folgenden Quellcode ein, compilieren Sie ihn und erstellen Sie ein ausführbares Programm. 
  #include <stdio.h>
  #include <math.h>
  
  int main (void)
  {

     int i;

     printf("Dieses Programm gibt alle Quadratzahlen zu den Basen 1 bis 20 aus!");
     printf("\n\n");

     for ( i = 1 ; i < 21 ; i=i+1 )  /* for-Schleife wird 20-mal durchlaufen */
     {
     /* for-Schleife Anfang */

        printf("Basis: %d", i);
        printf("\t");                /* Tabulator */
        printf("Ergebnis: %d",i*i);  /* Berechnung der Quadratzahl
                                        in der printf()-Funktion! */
        printf("\tASCII: %c", (char) ('A'+i));
        printf("\tpow: %f", pow(i, 2));
        printf("\n");
      }
      /* for-Schleife Ende */

      return 0; 
  }
Was gibt das Programm in den Spalten "ASCII" bzw. "pow" aus? Was bewirkt das Zeichen '\t' bei der Ausgabe?

Schreiben Sie das Programm mit den C++-Befehlen für Ein- und Ausgabe um. Erkennt Ihr Compiler die Vorgaben des C++-Standards an oder beharrt er noch auf veralteten Konventionen zum Einbinden von Headerfiles?

Wie muß man das Programm ändern, damit es zuerst die Quadratzahl von 20, dann die von 19, ... und dann die von 1 ausgibt?

Abzugeben ist die Ausgabe des ursprünglichen Programmes und der Quellcode des Files mit der C++-Ausgabe und der veränderten Reihenfolge der for-Schleife.

Aufgabe 03

keine Korrektur der Aufgabe, Besprechung in den Übungen

Erstellen Sie ein Programm, das
  1. den durchschnittlichen Benzinverbrauch auf 100 km und
  2. die Kosten pro gefahrenen Kilometer berechnet.
Die Eingabeparameter (float-Zahlen) sind
  • die gefahrenen Kilometer,
  • der gesamte Verbrauch in Litern und
  • der Benzinpreis.
Orientieren Sie sich beim Schreiben des Programmes an dem Vorlesungsbeispiel zur Berechnung der Hypothenuse und der Fläche eines rechtwinkligen Dreiecks.

Was meldet der Compiler, wenn Sie sich beim Funktionsaufruf "printf()" oder beim Namen des Ausgabeobjekts "cout" vertippen?
Was meldet der Compiler, wenn Sie die Variablen versehentlich vom Datentyp "real" anlegen (anstatt vom Datentyp "float")?
Erhalten Sie zusätzliche Informationen durch eine Fehlernummer oder eine Zeilennummer?
