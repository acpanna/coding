Aufgabe 06

Lösung per e-mail an hiwi511@btcipmatx1.cip.uni-bayreuth.de (Matthias Kohl)

Schreiben Sie ein C-/C++-Programm mit einer for-Schleife, das die Fibonacci-Zahlen für n = 0, 1, ..., 10 berechnet und die Werte ausgibt.
Die Fibonacci-Folge wird dabei rekursiv berechnet mit den beiden Startwerten
a0 = 1
a1 = 1
und der Rekursion
an = an-1 + an-2
für n >= 2.

Wieviele Variablen brauchen Sie mindestens? Hängt die Zahl von der maximalen Iterationszahl n ab?

Abzugeben ist das C-/C++-Programm und eine schriftliche Beantwortung der Fragen.
